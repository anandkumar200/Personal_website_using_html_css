# Personal Project (Capstone Project 2)

A clean, responsive personal portfolio website built as part of The Complete 2026 Web Development Bootcamp on Udemy by Angela Yu. 

> ⏱️ **Challenge Context:** This project was built entirely from scratch in under 4 hours as a timed Capstone challenge! While there are many areas I plan to polish and expand in the future, it serves as a snapshot of my core HTML and CSS layout skills.

---

## 🚀 Features

* **Clean Hero Section:** A welcoming introduction highlighting my background as an aspiring web developer.
* **About Me & Skills Layout:** Showcases my dual passions—Development and Cinema—using a clean CSS Grid layout.
* **Responsive Mindset:** Built using Flexbox and Grid alignments to handle content structure dynamically.
* **Integrated Design Elements:** Features dynamic GIFs and custom image placements for an engaging user experience.

---

## 🛠️ Tech Stack Used

* **HTML5:** Semantic markup structure (`<header>`, `<footer>`, `<section>` blocks).
* **CSS3:** Custom styling featuring Flexbox, CSS Grid layout properties, border-radius manipulation, and custom typography pairings.
